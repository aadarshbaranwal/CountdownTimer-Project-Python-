The Countdown Timer App is a Python GUI application built using Tkinter.
It allows users to set a countdown time in minutes and seconds, start the timer, pause it, reset it, and receive a sound alert when the countdown finishes.
The app also includes:
Quick preset buttons
Pause & resume functionality
Reset option
Toggleable Dark Mode
Cross-platform sound notification support
